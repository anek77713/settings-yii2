# Tests

1. Create `settings_yii2_tests` database and run

   ```
   composer test-prepare
   ```

2. Run tests

   ```
   composer test
   ```

3. Run tests with coverage

   ```
   composer test-coverage
   ```

   or

   ```
   composer test-coverage-html
   ```

4. Show coverage dashboard

   ```
   composer test-coverage-html-open
   ```
