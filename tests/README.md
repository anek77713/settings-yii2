# Tests

## Preparation

Create `settings_yii2_tests` database and run

```
composer test-prepare
```

## Commands

- Run tests
  ```
  composer test
  ```

- Run tests with coverage
  ```
  composer test-coverage
  ```

   or

  ```
  composer test-coverage-html
  ```

- Show coverage dashboard
  ```
  composer test-coverage-html-open
  ```
